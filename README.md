# iroha

## Getting Started

### Installation

`pom.xml`
```xml
 <repository>
  <id>github</id>
  <name>GitHub Packages</name>
  <url>https://maven.pkg.github.com/2mug1/iroha</url>
</repository>

<dependency>
  <groupId>net.iamtakagi</groupId>
  <artifactId>iroha</artifactId>
  <version>1.0.0</version>
</dependency>
```
